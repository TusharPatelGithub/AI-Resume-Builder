# AI Resume Builder (CRA + Tailwind)

A clean starter that lets users enter resume data, preview it live, and export a polished PDF via **@react-pdf/renderer**. 
Firebase (v12) is set up for optional auth/storage — add your keys in `.env`.

## Quick start
```bash
npm i
npm start
```

## Environment
Create a `.env` in the project root with:
```
REACT_APP_FIREBASE_API_KEY=
REACT_APP_FIREBASE_AUTH_DOMAIN=
REACT_APP_FIREBASE_PROJECT_ID=
REACT_APP_FIREBASE_STORAGE_BUCKET=
REACT_APP_FIREBASE_MESSAGING_SENDER_ID=
REACT_APP_FIREBASE_APP_ID=
```
These are optional if you won't use Firebase yet.

## Build
```bash
npm run build
```
